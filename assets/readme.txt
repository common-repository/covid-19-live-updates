Covid 19 Live Updates Widget

== Description ==

Simply show latest updates on Confirmed Cases, Deaths and Recovered.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/covid-19-live-updates` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Widgets to add and configure Plugin in the widgets 

== Frequently Asked Questions ==

= Why I need this plugin =

To give your visitors quick info about Covid19 Situation
